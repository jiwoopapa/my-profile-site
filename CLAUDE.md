# 프로젝트 개요
바닐라 JS + CSS + HTML 기반 개인 프로필 사이트

# 기술 스택
- HTML5 (시맨틱 마크업)
- CSS3 (Flexbox/Grid 레이아웃)
- 바닐라 JavaScript (ES6+)
- 빌드 도구 없음 (순수 정적 파일)

# 프로젝트 구조
- `index.html` : 메인 페이지
- `style.css` : 전체 스타일
- `script.js` : 인터랙션 로직

# 코드 컨벤션
- JS 변수/함수명: camelCase
- CSS 클래스명: kebab-case
- 들여쓰기: 2칸 스페이스
- JS 모듈 시스템 없이 전역 스코프 최소화 (IIFE 또는 이벤트 리스너 내부 처리)
